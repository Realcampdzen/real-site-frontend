## 🔄 Как работает система

### Схема деплоя:

```
┌─────────────────────────────────┐
│  Локальная разработка           │
│  (npm run dev)                  │
└──────────────┬──────────────────┘
               │
               │ git push origin main
               ▼
┌─────────────────────────────────┐
│  GitHub Actions                 │
│  (Автоматически)                │
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│  GitHub Pages (Staging)         │
│  ✅ Полная версия               │
│  ✅ Все тестовые функции        │
│  ✅ Недоработанные разделы      │
│  URL: realcampdzen.github.io/   │
└─────────────────────────────────┘
               │
               │ После тестирования
               ▼
┌─────────────────────────────────┐
│  prepare-deploy.ps1             │
│  (Ручной запуск)                │
└──────────────┬──────────────────┘
               │
               ▼
┌─────────────────────────────────┐
│  deploy-ready/                  │
│  ✅ Production версия           │
│  ✅ Только готовые функции      │
└──────────────┬──────────────────┘
               │
               │ Загрузка на NIC.RU
               ▼
┌─────────────────────────────────┐
│  NIC.RU (Production)            │
│  ✅ Готовая версия              │
│  ✅ Только для клиентов         │
│  URL: real-vibe.studio          │
└─────────────────────────────────┘
```

## 📝 Следующие шаги

### Немедленно доступно:

1. **Деплой на GitHub Pages:**
   ```bash
   git add .
   git commit -m "Описание"
   git push origin main
   ```
   → Автоматически задеплоится на staging

2. **Деплой на NIC.RU:**
   ```powershell
   .\prepare-deploy.ps1              # Подготовить файлы
   .\list-files-to-upload.ps1        # Получить список
   ```
   → Затем загрузите файлы **по одному** в соответствующие папки на сервере
   → См. `NIC_RU_UPLOAD_GUIDE.md` для подробной инструкции

### Когда будете готовы:

3. **Настройка исключения тестовых функций:**
   - Откройте `prepare-deploy.ps1`
   - Найдите секцию `# PRODUCTION: Remove test features`
   - Добавьте логику исключения:
     - Недоработанных разделов
     - Тестовых ботов
     - Placeholder изображений
